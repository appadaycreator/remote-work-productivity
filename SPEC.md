# テレワーク生産性診断【無料】無料Webツール - 技術仕様書

## 概要

**サービス名**: Remote Work Productivity
**バージョン**: 1.3.0（M3/M4実装版）
**更新日**: 2026-06-10
**URL**: https://appadaycreator.com/remote-work-productivity/

在宅環境・集中力・コミュニケーション頻度からテレワーク生産性を診断して改善点を提案。登録不要・完全無料でご利用いただけます。

## 更新履歴

### v1.3.0 (2026-06-10)
**実装内容（M3・M4）**:
- localStorage による診断結果の自動保存・履歴管理機能
- Chart.js によるレーダーチャート（カテゴリ別スコア比較）
- スコア推移グラフ（過去5回まで追跡）
- 履歴一覧表示・個別削除・全削除機能
- スタート画面に「前回の診断」プレビュー機能
- 結果画面にカテゴリ別スコア可視化

### v1.2.0 (2026-06-05)
**改善内容**:
- 結果タイプの判定ロジック改善: スコアレンジ (0-5, 6-8, 9-11, 12-15) で細分化
- 結果説明の最適化: 各タイプごとに「症状・原因・改善方法」で構造化
- モバイルUI最適化: シェアボタンを bottom fixed で常に見えるように
- 「もう一度診断」ボタンをモバイルで結果の直下に配置（スクロール軽減）
- 横向き (landscape) 対応: min-height: 600px での最適化
- タブレット (768px以上) での 5列カテゴリ表示

### v1.1.0 (2026-06-03)
**追加機能**:
- スコア表示機能: 総合スコア (0-15点) + ゲージバー + カテゴリ別★表示
- 動的シェア機能: 診断結果スコアを含むSNS共有テキスト
- 結果画像保存機能: html2canvas で診断結果を PNG でダウンロード
- 詳細表示トグル: モバイル向けに説明文の表示/非表示を切り替え可能

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: 診断完了時に自動保存、ページ読み込み時に自動復元
- **履歴保持**: 最新10件まで保持、新規診断時に古いものを自動削除
- **クリア**: 個別削除・一括削除ボタンで管理、またはブラウザのサイトデータ削除で初期化
- **保存データ**: スコア、結果タイプ、カテゴリ別スコア、診断日時

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 使い方

1. ページを開き、診断の説明を確認する
2. スタートボタンをクリックして診断を開始する
3. 表示される質問に順番に回答する（約1〜2分）
4. すべて回答すると診断結果が自動表示される
5. 結果のアドバイスを確認してSNSでシェアする

## よくある質問（FAQ）

**Q: テレワーク生産性診断は無料で使えますか？**

はい、完全無料でご利用いただけます。登録・ログイン・アプリのインストールも不要です。

**Q: 何回でも診断できますか？**

はい、何度でもご利用いただけます。条件を変えて繰り返し診断してみてください。

**Q: スマートフォンでも使えますか？**

はい、スマートフォン・タブレット・PCすべてのデバイスに対応しています。

**Q: 診断結果は保存されますか？**

結果はブラウザには保存されません。スクリーンショットやSNSシェアでご記録ください。

**Q: 入力したデータはサーバーに送信されますか？**

いいえ。すべての処理はブラウザ内で完結し、個人情報・入力データのサーバー送信は行いません。
